## 🚀 КАК НАЧАТЬ РАЗРАБОТКУ

### Шаг 1: Подготовь GitHub (5 минут)

```bash
# 1. Создай папку проекта
mkdir family-nutrition-app
cd family-nutrition-app

# 2. Инициализируй Git
git init

# 3. Скопируй все файлы из outputs в эту папку
# (CLAUDE_INSTRUCTIONS.md, CURRENT_TASK.md, etc.)

# 4. Первый коммит
git add .
git commit -m "Initial commit: Project setup and documentation"

# 5. На GitHub создай новый репозиторий
# Потом добавь remote и пушни:
git remote add origin https://github.com/USERNAME/family-nutrition-app.git
git branch -M main
git push -u origin main
```

**Инструкции подробнее → GITHUB_SETUP.md**

---

### Шаг 2: Начни разработку в Claude Code (5 минут на сессию)

```
1. Открой Claude Code

```
